# my_quotes_example.txt
# ─────────────────────────────────────────────────────────────────────────────
# Example plain-text quote file for Quote Screensaver.
#
# FORMAT RULES:
#   • One quote per line.
#   • Blank lines and lines starting with # are ignored.
#   • To add an author, end the line with:  -- Author Name
#     (space, two dashes, space, then the author)
#   • Quotes without -- are shown without attribution.
#   • Save the file as UTF-8 for best results.
#
# See docs/QUOTE_FORMATS.md for the full format reference.
# ─────────────────────────────────────────────────────────────────────────────
